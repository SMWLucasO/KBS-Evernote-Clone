PlantUmlClassDiagramGenerator: https://github.com/pierre3/PlantUmlClassDiagramGenerator
PlantUML: http://plantuml.com/

Install PlantUmlClassDiagramGenerator to generate *.puml files from project: 
$ dotnet tool install --global PlantUmlClassDiagramGenerator --version 1.2.0
Download PlantUML to generate *.png files from *.puml: http://plantuml.com/download

Generate *.puml files (add -createAssociation for associations and/or add -allInOne to generate whole class diagram to 1 include.puml file):
$ puml-gen C:\git\KBS-Evernote-Clone\EvernoteClone\ C:\git\KBS-Evernote-Clone\UML\ -dir -excludePaths EvernoteCloneLibraryTests\,EvernoteCloneLibrary\obj\,EvernoteCloneLibrary\Properties\,EvernoteCloneGUI\obj\,EvernoteCloneGUI\Properties\,packages\,.idea\,.vs\

Generate *.png from all *.puml files: 
$ java -jar plantuml.jar "C:\git\KBS-Evernote-Clone\UML\**.puml"
